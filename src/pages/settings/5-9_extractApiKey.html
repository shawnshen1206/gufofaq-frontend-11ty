如何呼叫萃取 API

以 X-API-Key 標頭呼叫；mode 1=國家、2=時間、3=兩者；text 長度 2–5000 字；額度用罄回 429。

curl -X POST "https://<your-product-host>/extract/country-or-time" \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"我們下週三在東京開會","mode":3}'
